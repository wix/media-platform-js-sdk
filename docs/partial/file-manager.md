### getUploadUrl()

```javascript
fileManager.getUploadUrl({
  mimeType: 'Content-Type: image/gif',
  path: '/path/to',
  size: 12345, // in bytes
  acl: 'public'
})
.then(response => {
  console.log(response.uploadToken, response.uploadUrl);
});
```
